A comprehensive implementation of Decision Tree Classifier on the famous Iris dataset. This project demonstrates the complete machine learning pipeline including data exploration, preprocessing, model training, evaluation, and visualization.

## 📊 Project Overview

This project implements a Decision Tree classifier to classify iris flowers into three species:
- **Setosa**
- **Versicolor**
- **Virginica**

Based on four features:
- Sepal length
- Sepal width
- Petal length
- Petal width

## 🎯 Key Features

- ✅ Data exploration and visualization
- ✅ Correlation analysis
- ✅ Label encoding for categorical variables
- ✅ Train-test split (80:20 ratio)
- ✅ Decision Tree classifier implementation
- ✅ Model evaluation metrics
- ✅ Confusion matrix visualization
- ✅ Decision tree visualization
